## ğŸš€ Getting Started

### Prerequisites
- Node.js 20+
- Java 17+
- Docker & Docker Compose
- PostgreSQL 15+

### Local Development

1. **Clone the repository**
```bash
git clone https://github.com/vinayredee/jansevak.git
cd jansevak
```

2. **Run with Docker Compose**
```bash
cd deployment
docker-compose up
```

3. **Access the application**
- Frontend: http://localhost:5000
- Backend API: http://localhost:8080

### Manual Setup

**Frontend:**
```bash
cd frontend
npm install
npm run dev
```

**Backend:**
```bash
cd backend
mvn spring-boot:run
```
